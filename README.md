\# AI Resume Screening Tool



A full-stack MERN application that analyzes resumes against job descriptions using AI and gives a match score with feedback.



\## Features

\- Google Sign-in authentication

\- Upload resume (PDF) and paste job description

\- AI-powered resume analysis using Cohere AI

\- View analysis history

\- Admin panel to view all users' submissions



\## Tech Stack

\- Frontend: React, Vite, Material UI

\- Backend: Node.js, Express

\- Database: MongoDB

\- AI: Cohere AI (Command model)

\- Auth: Firebase Google Sign-in



\## Run Locally
